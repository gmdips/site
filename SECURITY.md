# 🔒 Kebijakan Keamanan — GDIPS Website

## Melaporkan Vulnerability

Jika kamu menemukan kerentanan keamanan di website GDIPS, **JANGAN buka public issue.**

Laporkan via:
- **Email:** fless@gdi.ps
- **Discord DM:** @fless di [server Discord](https://discord.gg/YyeZ2Sxjgf)
- **GitHub Security:** [Private Reporting](https://github.com/gmdips/site/security/advisories/new)

## Jenis Vulnerability yang Relevan

- XSS (Cross-Site Scripting)
- CSRF (Cross-Site Request Forgery)
- Content Injection
- Open Redirect
- Information Disclosure
- DLL

## Respons

- **Acknowledgment:** 48 jam
- **Assessment:** 7 hari
- **Fix:** Tergantung severity

## Credit

Reporter akan diberi credit di changelog (kecuali minta anonim).
